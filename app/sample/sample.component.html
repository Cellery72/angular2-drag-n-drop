These are simple drag-n-drop examples I created using ng2-dragula. Examples are great and all, but the way I learned best was my creating some of my own (these ones)!

You can move these elements between these two containers
Moving them anywhere else isn't quite possible
There's also the possibility of moving elements around in the same container, changing their position
This is the default use case. You only need to specify the containers you want to use
More interactive use cases lie ahead
Make sure to check out the documentation on GitHub!
